# WhiteroomPerception

# Description
A small, first person exploration/puzzle game where perception changes as the player collects items in a white room, uncovering more access to vision to enable them.

# Controls
Mouse - Look Around |
Interaction - Colliding with Object |
WASD - Move
> W - Strafe Forward |
> A - Strafe Left |
> S - Strafe Backward |
> D - Strafe Right |


# Dev Log
Stage based environment toggling using script to replace visible environments upon collision with trigger item.
Dynamic lighting intensity.
Moving projectiles, rudimentary as I had to figure it out myself :( 
Hostile projectiles that restart the game.
Maze with potential need to backtrack.

# Assets Used
No external assets were used, as everything was made using Cubes.
